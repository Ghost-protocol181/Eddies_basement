# Eddie's Basement Beta Test

## Five-minute tester script

1. Open the homepage on your normal phone or computer.
2. Confirm games appear without refreshing.
3. Search for a game you know.
4. Try one platform or setup filter, then Clear All.
5. Open a game and save it as a favorite.
6. Close it and confirm the game appears in Recently Viewed.
7. Use Eddie, Choose and pick a randomizer mode.
8. Pick Again once and confirm the same game is not immediately repeated.
9. Open one game link.
10. Submit anything confusing through Beta Feedback.

## Report these immediately

- No games appear after 15 seconds.
- Eddie, Choose is missing or does nothing.
- A broken-image icon appears.
- The same placeholder appears throughout the first rows.
- Search or Clear All stops working.
- A game card opens the wrong game.
- A game marked browser-only requires a download.
- A game marked free is no longer free.
- Buttons are cut off or impossible to tap.
- The page causes horizontal scrolling on a phone.

## Devices to cover

- Current iPhone Safari
- Older/smaller iPhone Safari
- Android Chrome at approximately 360px width
- Chrome on Windows or macOS
- Firefox desktop
- Edge desktop
- Private browsing with third-party requests restricted
- Slow mobile connection

## Release decision

Do not expand the feature set during beta unless a missing feature blocks the core job: finding and choosing a multiplayer game. Fix stability, wrong information, broken artwork and confusing decisions first.
